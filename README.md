# PolyWatch Dashboard

Single-file HTML dashboard for PolyWatch — whale copy-trading on Polymarket.

Deployed via Cloudflare Pages. Backend API lives at
https://polywatch-api-production.up.railway.app

## Files
- `index.html` — the dashboard (Cloudflare Pages default entry)
- `polywatch-v4.html` — identical copy under the versioned name

No build step. Cloudflare Pages serves `index.html` directly.
